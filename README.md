# 可视化的发送HTTP请求小工具

#### 介绍
可视化的发送HTTP请求小工具，方便做测试时用

#### 软件架构
1. 一个主界面用于填写请求URL和一些参数，并显示响应结果。
2. 可以把用过的请求保存成XML文件供下次加载使用。
3. 支持Get和Post方式发送请求。

#### 安装教程

下载解压，用NetBeansIDE打开编译，运行即可。
